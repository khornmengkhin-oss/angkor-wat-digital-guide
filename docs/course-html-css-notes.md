# Simple HTML and CSS Notes

These notes are only a small reminder for our group. They are based on the class slides we used for this project.

## HTML We Can Use

- `<!DOCTYPE html>` tells the browser this is HTML5.
- `html`, `head`, and `body` make the basic page.
- `meta charset="UTF-8"` helps the page show text correctly.
- `title` shows the browser tab name.
- `h1` to `h6` are headings.
- `p` is for paragraph text.
- `br` can make a line break if we really need it.
- `ul`, `ol`, and `li` are for lists.
- `a` is for links.
- `img` is for images, and we should add `alt`.
- `header`, `nav`, `main`, `section`, `article`, and `footer` help make the page meaning clear.

## CSS We Can Use

- External CSS files are linked in the `head`.
- Element selectors style tags like `p` or `h1`.
- Class selectors start with a dot, like `.button`.
- ID selectors start with `#`, but we should not use too many.
- Common properties: `color`, `background-color`, `font-size`, `font-family`, `margin`, `padding`, and `border`.
- We can use `display: flex` for one direction layout.
- We can use `display: grid` for rows and columns.
- We can use `@media` for smaller screens.
